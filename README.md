# Ecommerce-Sales-Analysis-Dashboard

<img width="950" height="577" alt="image" src="https://github.com/user-attachments/assets/8e7bf7c7-a2f8-4fb2-ad04-49f5def39b2c" />


A comprehensive Microsoft Excel dashboard that transforms raw ecommerce sales data into actionable business insights using Pivot Tables, Pivot Charts, KPI Cards, Geographic Maps, and Interactive Visualizations.

---

## Project Overview

This project analyzes nearly **10,000 ecommerce transactions** and presents important business metrics through an interactive dashboard.

The dashboard enables business users to monitor sales performance, profitability, regional trends, category performance, and year-over-year growth without requiring advanced technical knowledge.

It demonstrates how Excel can be used as a Business Intelligence (BI) tool for data-driven decision making.

---

## Dataset

The dataset contains **9,994 sales records** with **22 attributes**, including:

- Order ID
- Customer Information
- Order Date
- Shipping Details
- Product Category
- Product Name
- Region
- State
- Sales
- Quantity
- Discount
- Profit

## Dashboard Features

### KPI Dashboard

Displays important business metrics including:

- Total Sales
- Total Profit
- Order Count
- Quantity Sold
- Profit Margin

---

### Year-over-Year Analysis

Compares business performance across multiple years.

Metrics include:

- Sales Growth
- Profit Growth
- Orders
- Quantity Sold
- Profit Margin

---

### Monthly Performance

Interactive charts showing:

- Monthly Sales
- Monthly Profit
- Sales Trend
- Profit Trend

---

### Category Analysis

Analyze performance across product categories:

- Furniture
- Office Supplies
- Technology

Visualized using:

- Pie Chart
- Waterfall Chart

---

### Regional Analysis

State-wise sales analysis using:

- Geographic Map
- Regional Sales Comparison

Helps identify:

- High-performing regions
- Low-performing regions
- Market opportunities

### Product Performance

Top-selling product categories are visualized using:

Useful for identifying:

- Best-selling products
- Revenue-driving categories
